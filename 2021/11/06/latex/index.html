# Latex

latex 命令多采用 \ []{} 的格式

基本结构

\document[UTF8]{ctexart}

\begin{document}

\end{document}

# 字体

  • \textbf {文本内容} : 加粗文本内容
  • \textit {文本内容} : 设置斜体
  • \underline {文本内容} : 设置下划线

# 章和节

  • 切换段落需要输入两个换行符 也就是按两次回车 一个换行符仅仅会显示一个空格
  • \part {部名称} : 表示书籍中的第几部
  • \chapter {章节名称} 通常用于表示章节中的第几章，大于 chapter
  • \section {章节名称} : 开启一个新的章节
  • \subsection {章节名称} : 创建下一级的章节，要创建更下一级的章节就多加个 sub

# 图片

  • 第一步要先引用包 \usepackage{grathicx}

  • \includegraphics [可选参数]{图片文件名字} : 在当前位置添加一张图片 可以省略图片后的扩展名称

    1. width=0.5\textwidth :

  • 给图片添加标题

    \begin{figure}    放在figure中
\centering   居中显示
\includegraphicx {head}
\caption{哈哈哈}       标题名称
\end{figure}

# 列表

  • 切换到列表中的环境

  • 无序列表:

    \begin {itemize}
\item 列表项1
\item 列表项2
\end{itemize}

  • 有序列表

    \begin {enumerate}
\item 列表项1
\item 列表项2
\end{enumerate}

# 数学公式

  • 行内公式: $ $ 在行内直接显示公式

  • 单独显示一行公式

    \begin{equation}

\end{quation}

可以简写为
\[

\]

  • 公式常用指令

    • \over : 表示分数 分子在前 分母在后
    • \varphi : 小写的 φ
    • \phi : 大写的 φ

    可以使用在线公式编辑器和 mathpix

latex.codecogs.com/eqneditor/editor.php 在线公式编辑器 也可以快速查阅公式的语法

# 表格

\begin{tabular} {传入参数|参数} c表示居中 l表示靠左 多少个字母就表示多少列    p{列宽} 单独指定每一列的宽度
\hline
1 & 2 ^ 3 \\
\hline    水平方向边框
1 & 3 & 2 \\      每一列数据用&分割
				每一行的数据用\\分割
				
\caption{表格的标题}
\end{tabular}
