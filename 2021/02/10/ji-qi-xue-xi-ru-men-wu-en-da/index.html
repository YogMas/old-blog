# 初步介绍机器学习

机器学习可以分为两类

  • 监督学习 supervised learning 人为教机器
  • 不监督学习 unsupervised learning 机器自己学习

# 监督学习 Supervised learning

给出一定的输入值和输出值 (正确答案), 让机器通过另一些输入值来预测输出值

  • 回归问题 (regression problem):. 预测一个连续值输出(real-valued)

    Eg: 就像 f (x) 中通过给定一个值 s 可以得到一个值 f(s)

  • 分类问题 (classification problem): 预测一个离散值输出(Discrete-valued)

    Eg: 输出 0 和 1 , 1 代表 True 0 代表 False

根据数据集中的一系列输入数据，通过算法预测，得出一系列的输出数据 (正确答案).

# 无监督学习 Unsupervised learning

没有给定正确答案，通过算法让机器自己将输入的数据分成不同的类型

  • 聚类算法 (cluster): 将相同类型的数据化作一个族里面

  • 鸡尾酒会问题：给定混合信号，如何分离出鸡尾酒会中同时说话的每个人的独立信号

# 梯度下降

假设有函数 F(x,y) , 先给定一个 x 和 y 的值，再不断改变 x 和 y 的值，直到找到 F(x,y) 的最小值或者局部最小值

形象说明：在这样的三维图像上从小山的顶端或者半山腰出发，找到一个最近的路径到达低谷，每运行一次梯度下降算法，都会下降一点高度，也就是 J函数 的值会小一点

  • α: 学习率，表示了我们每次下山时候迈出步子的大小，其值越大下降越迅速，如果学习率过大得的话，有可能越过最低点

# 线性回归下的梯度下降

线性回归的函数总是一个凸函数 (没有局部最优解，只有一个全局最优解)

Batch 梯度下降：每次梯度下降的时候，我们都使用了整个样本集 ( J函数 中的求和符号)
