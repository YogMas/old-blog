  4. 移动设备操作系统

# Unix

伴随着 B 语言和 C 语言的由来，同时由于其创作者可以直接访问使用该系统的电脑，电气公司将其私有化后随之产生了 Minix

# Minix

Unix 私有化之后，大学无法使用 Unix 系统进行教学，就出现了 Minix

# Linux

Minix 上可以实现的功能非常少，为了达到相应的功能需要，就出现了 Linux

# Linux 内核

# Linux 发行版

在 Linux 内核的基础上包括了桌面环境，办公套件，数据库等应用软件

# 创建和删除操作

  1. touch

    • 没有该名的文件存在，则创建一个新的文件
    • 有改名字的文件存在，则更改文件的末次修改日期

  2. mkdir

    -p 连续创建多个目录

    mkdir -p a/b/c/d

  3. rm

    -r 连续删除多个同一文件夹下的文件

    -f 强制删除无提示信息

    ls 的通配符对 rm 一样适用

# 拷贝和移动命令

  1. tree

    以树状图列出文件目录结构

    -d 只显示目录

  2. cp 源文件 目标文件

    -i 覆盖文件前提示

    -r 复制整个目录

  3. mv 源文件 目标文件

    -i 覆盖前提示
