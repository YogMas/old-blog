# Git

# 常用指令

  • git add 文件名 ：添加到缓存区
  • git commit -m "提交说明" : 提交到本地代码库
  • git init : 初始化 git 仓库
  • git status : 时刻掌握仓库当前的状态
  • git diff : 查看仓库更改中的不同
  • git log （--pretty=online） : 显示从最近到最远的提交日志 后面的参数可以简化显示
  • git reset --hard HEAD^ (commit ID) : 文件版本回退
  • git reflog : 显示你之前的每一次命令
  • git checkout -- file : 把 file 文件在工作区的修改全部撤销，原理是用版本库里的版本替换工作区的版本
  • git reset HEAD <file> : 把暂存区的修改撤销掉，重新放回工作区
  • git rm : 删除文件，执行后再 git commit 才能从版本库中删除

# 基本知识

git 中使用 HEAD 来表示当前版本 HEAD^ 来表示上一个版本 HEAD^^ 来表示上上个版本 还可以写作 HEAD~100 来表示前一百个版本
